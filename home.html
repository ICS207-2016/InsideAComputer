Parts and Their Functions

ROM

The ROM (read-only memory) stores programs and software instructions in electronic devices as well as performing major input and output tasks. It is a type of memory storage system that can only be read and not changed.

RAM

Random Access Memory (RAM) allows information to be stored and retrieved on a computer. This temporary memory is not saved when the computer is turned off. RAM is measured in megabytes and gigabytes.

Network Card

The network card allows your computer to communicate over a network and access the Internet. It can connect to the internet with an ethernet cable or through a wireless connection (Wi-Fi).

Hard Drive

The hard drive stores all the information on your computer. This data is permanent, meaning it wonâ€™t be lost when the computer is turned off.

Heating Sink

A heat sink transfers the heat generated by an electronic or a mechanical device in a computer to a fluid medium, often air or a liquid coolant, where it is taken away, neutrailizing the computers temperature in order to prevent overheating.

Power Supply

The power supply converts the power from the wall outlet to the type of power needed by the computer in order to run. It sends power through cables to different components of the computer.

Floppy Disk Drive

The floppy drive is a piece of computer hardware that reads data from, and writes data to, a small disk.

Fan

The fan prevents the CPU from overheating. Cooling fans bring cool air into the computer, transfer hot air out of the computer.

CPU

The CPU runs a series of stored instructions (also known as a program). This program will take input from an input device, process the input and output the results to an output device.

Video Card

A video card (also known as a graphics card) is connected to the motherboard and generates what you see on the computer. Video cards include a processing unit, memory, a cooling mechanism and connections to a display device.

Video Card Slot

RAM Slot

RAM slot is what allows computer memory (RAM) to be inserted into the computer.

CD Slot

Ports

A port is a hole or connection found on the front or back of a computer which connect external devices to the computer.
